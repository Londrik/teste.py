🚀 Streamlit & Python Lab

Este repositório foi desenvolvido com o objetivo de centralizar e organizar os meus projetos de desenvolvimento, scripts de automação, resoluções de desafios de lógica e notebooks de aprendizagem em Python e Streamlit.

A estrutura foi totalmente remodelada para seguir as melhores práticas de organização de código e versionamento com o Git.

📂 Estrutura do Repositório

O projeto está dividido de forma temática nas seguintes pastas:

streamlit/
│
├── 📂 Aulas/        # Notebooks (.ipynb) e scripts com conteúdos teóricos e práticos das aulas.
├── 📂 Desafios/     # Exercícios de lógica de programação e desafios propostos.
├── 📂 Jogos/        # Pequenos projetos de jogos desenvolvidos no terminal ou interface gráfica.
└── 📂 Projetos/     # Aplicações completas, incluindo sistemas em Streamlit e scripts modulares.


🛠️ Tecnologias e Ferramentas Utilizadas

Linguagem Principal: Python 3

Interface Web: Streamlit

Análise & Aprendizagem: Jupyter Notebook (.ipynb)

Editor de Código: Visual Studio Code (VS Code)

Ambiente Operacional: Linux (Mint / Arch Linux)

💻 Como Executar as Aplicações

Se quiser clonar este repositório e executar os projetos locais na sua máquina, siga os passos abaixo:

1. Clonar o Repositório

git clone [https://github.com/Londrik/teste.py.git](https://github.com/Londrik/teste.py.git)
cd teste.py


2. Criar e Ativar um Ambiente Virtual (Opcional, mas recomendado)

python -m venv .venv
# No Linux:
source .venv/bin/activate
# No Windows (PowerShell):
.venv\Scripts\Activate.ps1


3. Instalar as Dependências

(Caso adicione um ficheiro requirements.txt no futuro)

pip install -r requirements.txt


4. Executar um Projeto Streamlit

Para os ficheiros contidos na pasta Projetos/ que utilizam a interface do Streamlit (como o index.py), utilize:

streamlit run Projetos/index.py
